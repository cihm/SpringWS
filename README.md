# SpringWS

##Linux env

  * Clone project
   <br />
 $ git clone https://github.com/cihm/SpringWS.git 
   <br />
  * Modify your deploy path
   <br />
  $ vim gradle.properties 
    <br />
  * Run gradle  
    <br />
 ./gradlew build
   <br />
 ./gradlew deploy
